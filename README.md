1.どんな処理を実装したのか
--------------
はじめに、PCのカメラで動画を撮影するコードを書いた。
撮影の終了は[q]を押したタイミングとした。  
そして、撮影された動画を"output.mov"として保存した。  
次に、撮影された動画から、1フレームごとに画像を出力させた。  
最後に、それぞれの画像から輝度値の平均を計算し、縦軸が輝度値、横軸が時間のグラフをプロットした。  
 
2.依存ライブラリとバージョン
-------------
openCV(4.1.0),numpy(1.16.2),matplotlib(3.0.3)
